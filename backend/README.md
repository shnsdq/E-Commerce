

### **E-Commerce Web Application**

**Tech Stack:** Node.js, Express.js, MongoDB, Cloudinary, JWT, Multer


**Description:**
Developed a full-featured eCommerce web application with secure user authentication, product management, shopping cart functionality, and Stripe payment integration.

**Key Features:**

* **User Authentication & Authorization:** Implemented JWT-based login/signup system with protected routes and role-based access (Admin/User).
* **Product Management:** Admin panel to add, update, delete products with multiple image uploads using Multer and Cloudinary.
* **Dynamic Cart & Order System:** Users can add products to cart, manage item quantity/sizes, and place orders.
* **Payment Integration:** Integrated **Stripe API** for secure payment processing and transaction handling.
* **Database:** Used MongoDB with Mongoose for schema design, relationships (Users, Products, Orders), and efficient querying.
* **RESTful APIs:** Built secure APIs using Express.js for all backend operations.
